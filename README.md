🏹 AI Wumpus World SolverAn interactive, visually rich simulation of the classic Wumpus World environment. This project features a fully autonomous AI agent equipped with a custom deductive reasoning engine. Watch in real-time as the AI calculates hazard probabilities, maps safe routes, takes calculated risks, and hunts the Wumpus.✨ Features🧠 Advanced AI LogicDynamic Knowledge Base: The agent dynamically maps the grid, storing visited states, sensor data (Breeze, Stench, Glitter), and calculating exact probabilities for Pits and the Wumpus.Propositional Deduction: Uses intersection logic to isolate the exact location of the Wumpus and definitively flag safe squares.Risk-Aware Pathfinding: Utilizes Breadth-First Search (BFS) to navigate guaranteed safe paths. When no safe path exists, it calculates the lowest-risk frontier to explore.Live Monologue: A real-time scrolling UI log details exactly what the AI is "thinking" and doing at every step.🎮 Interactive Visuals"God Mode" Overlay: See the hidden grid, animated particle effects for gold, flowing breezes around pits, and stenches radiating from the Wumpus.Probability Mapping: The AI's internal knowledge is overlaid on unvisited squares, showing real-time risk percentages (e.g., Risk: 33% or CERTAIN PIT).Directional Animation: The agent uses strict tank controls, visibly turning to face its target before moving or firing its bow.⚙️ Technical HighlightsWeb-Ready Architecture: Built using an asynchronous main loop (asyncio), making the codebase inherently compatible with WebAssembly ports like pygbag for browser-based play.Zero Dependencies (Mostly): Relies solely on PyGame and standard Python libraries (math, collections, random).🚀 Installation & UsagePrerequisitesYou will need Python 3.x installed on your machine.SetupClone the repository:git clone [https://github.com/YourUsername/wumpus-agent.git](https://github.com/YourUsername/wumpus-agent.git)

(Note: If you named your main python file differently, replace main.py with your file name).🕹️ ControlsYou have complete control over the simulation's pacing. You can let the AI run wild or step through its logic frame-by-frame.[Spacebar] - Toggle Auto-Play mode on/off.[ S ] - Manually trigger a single AI Step (highly recommended for watching the deduction engine work).[ R ] - Reset the board. This generates a completely new, mathematically solvable grid layout.[Mouse Scroll] - Scroll up and down through the AI's action log on the right sidebar.📖 The AI RulesetThe agent follows standard Wumpus World rules to maximize its score:+1000 for escaping alive with the Gold.-1000 for dying (falling into a Pit or being eaten by the Wumpus).-10 for shooting an arrow (the agent only has one).-1 for every movement or turn (encourages efficient pathfinding).How it survives: The AI prioritizes guaranteed safety. It will always use BFS to find the nearest 100% safe, unvisited square. If it detects a Stench and narrows the Wumpus location down to a single square, it will pathfind to an adjacent square, turn, and fire its arrow to clear the board. If no safe squares exist, it calculates a (Pit Probability * 2) + Wumpus Probability + Distance score to choose the mathematically safest risk.
